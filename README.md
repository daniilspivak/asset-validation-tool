# Asset Validation Tool

CLI tool for validating asset folder structure, naming conventions, and required files against a reference snapshot.

Built for internal game studio workflows to ensure assets strictly follow predefined structure and naming rules.

## Features

- Compares asset folders against a reference structure
- Detects missing, extra, and incorrectly named files
- Interactive CLI with persistent ignore rules
- Color-coded output for quick validation results

## Usage

Run `Run_AssetValidation.bat` and follow the on-screen instructions.

## Notes

User-specific settings are stored locally and are not part of the repository.
